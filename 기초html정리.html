차희망의 HTML 기초 정리

HTML : HyperText Markup Language

웹 페이지를 만들 때 사용 언어

HTML을 쓸때는 태그(tag)를 사용

태그의 종류와 용도

  • <h1> ~ <h6> : 제목 태그 - 숫자가 커질 수록 작아짐
  • <p> : 문단 태그
  • <ul> : 순서 없는 목록 태그
  • <ol> : 순서 있는 목록 태그
  • <li> : 목록 항목 태그
  • <a> : 링크 태그
  • <img> : 이미지 태그 - 사용 예: <img src="image.jpg" alt="이미지 설명">
  • <div> : 구역 태그
  • <span> : 인라인 구역 태그
  • <strong> : 강조 태그
  • <input> : 입력 필드 태그 - 사용 예: <input type="text" name="username" placeholder="사용자 이름">
  • <button> : 버튼 태그 - 사용 예 : <button type="button">버튼</button> - 간편한 버튼 구현

HTML 문서 구조

  1. <html> : HTML 문서의 루트 요소
  2. <head> : 문서의 메타 정보 포함
  3. <body> : 문서의 내용 포함
